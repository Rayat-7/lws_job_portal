# lws_job_portal
LWS Job Portal API powers a full job-hiring platform where Users (Job Seekers) can search and apply for jobs, and Company Admins can post and manage job listings.
The API provides endpoints for authentication, job listings, job details, applications, user profiles, company profiles, and utility data.
Its purpose is to enable a clean, structured backend for job search, job posting, application tracking, and profile management across both user roles.
